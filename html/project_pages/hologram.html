Interactive Holographic Kiosks

ROLE:
Software Developer

TOOLS:
JavaScript, HTML, CSS, NodeJs, Arduino


2021

Working for M1 Interactive, we created four holographic kiosks. My main responsibility on this project was writing the software that brings the whole experience together. For the kiosks with video and text content, I created UI overlays with HTML, CSS, and JavaScript. I also built a nodeJS server that communicated with the Arduino that controlled the robotics. The node server acted as the central hub that kept the UI content, audio, lighting, and user interaction in sync.
